Este repositorio implementa un conjunto de herramientas simbólicas para optimización matemática utilizando SymPy, 
centradas en el análisis de puntos críticos, optimización con restricciones y condiciones de optimalidad clásicas.
El código permite trabajar de forma simbólica con funciones de varias variables y cubre tres grandes bloques: 
optimización sin restricciones, multiplicadores de Lagrange y condiciones de Kuhn–Tucker (KKT).
